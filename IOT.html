Meus Projetos IoT

Projeto Carrinho Arduino

Um projeto completo de um carrinho seguidor de linha utilizando a plataforma Arduino. A atividade envolve a construção física do carrinho, a programação do microcontrolador Arduino para controlar os movimentos e sensores

Projeto Arduino Automação

Desenvolver um projeto completo de um Sistema de irrigação automática e lixeira com abertura automática utilizando a plataforma Arduino. A atividade envolve a construção física dos dois projetos, a programação do microcontrolador Arduino para controlar os movimentos e sensores, e a documentação detalhada do processo em um relatório técnico
